Overview

This interactive flexdashboard provides a comparative analysis of educational attainment levels between native-born and foreign-born populations across OECD member countries (plus Bulgaria and Romania). The dashboard enables exploration of education-migration intersections through interactive visualizations.

Dashboard Features

üìä Stacked Bar Chart

  • Compare native-born vs. foreign-born populations
  • Educational levels: Below upper secondary, Upper secondary, Tertiary
  • Country-specific analysis with aggregation over selected years

üåç Interactive World Map

  • Choropleth visualization of OECD + EU countries
  • Filter by birth place (native/foreign-born)
  • Filter by education level
  • Natural Earth projection for global context

‚öôÔ∏è Interactive Controls

Control Description
Country Select country for bar chart comparison
Birth place Native-born or Foreign-born filter for map
Education level Filter map by educational attainment
Year range Slider for temporal aggregation

Technical Implementation

Code 
# Core libraries
library(flexdashboard)
library(shiny)
library(dplyr)
library(plotly)
library(arrow)

# Data loaded from Parquet format (efficient columnar storage)
educ <- arrow::read_parquet("OECD_EducAtt.parquet")

# Interactive bar chart with plotly
plot_ly(
  data  = df_top,
  x     = ~BirthPlace,
  y     = ~Value,
  color = ~EducLevel,
  type  = "bar"
) %>%
  layout(barmode = "stack")

# Interactive choropleth map
plot_ly(
  df_map,
  type = "choropleth",
  locations = ~CountryCode,
  z = ~Value,
  locationmode = "ISO-3"
) %>%
  layout(
    geo = list(
      projection = list(type = "natural earth")
    )
  )
