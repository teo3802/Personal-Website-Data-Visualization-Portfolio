## Overview

This interactive Shiny application provides a comprehensive exploration of migration patterns across Austria's nine Bundesl√§nder (federal states). The dashboard combines demographic visualizations with geospatial mapping to reveal the distribution and composition of Austria's population with migration background.

## Dashboard Features

### üìä Dynamic Bar Charts & Pictograms

The application offers multiple visualization modes:

- **Bar charts** showing population distribution by various dimensions
- **Isotype pictograms** where each icon represents 100,000 people
- **Color-coded categories** for citizenship and origin countries

### üó∫Ô∏è Interactive Choropleth Map

- Hover over any Bundesland to see detailed statistics
- Population with migration background (1st and 2nd generation)
- Share percentages with color intensity encoding
- Emoji-based pictograms overlaid on the map

#| eval: false

# Core libraries
library(shiny)
library(dplyr)
library(ggplot2)
library(plotly)
library(sf)

# 1. Reactive filtering based on user input
# 2. Conditional rendering (pictogram vs bar chart)
